# docker_collect
collect docker compose

* mysql
* postgres
* redis
* mongodb
* wordpress
* php stack
* elasticsearch & kibana
* kafka
* rabbitMQ
* php-stack

- Use `docker pull portainer/portainer` to manage docker
```docker run -d -p 9000:9000 -v /var/run/docker.sock:/var/run/docker.sock portainer/portainer```
